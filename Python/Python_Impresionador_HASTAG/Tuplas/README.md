# Tuplas

## Estruturas

`tupla = (valor, valor, valor,...)`

## Diferença

Ela possui semelhança com a lista porém ela é imutável

### Vantagens

- é Mais eficiente em termos de perfomace
- Proteje a base de dados pois é imutavel
- É muito usada para dados heterogeneos
